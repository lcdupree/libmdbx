Detailed Description

Unmanaged cursor.

Like other unmanaged classes, cursor allows copying and assignment for instances, but does not destroys the represented underlying object from the own class destructor.

See also
mdbx_cursor_create()
mdbx_cursor_bind()
mdbx_cursor_close()
